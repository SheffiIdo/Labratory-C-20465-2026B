# Laboratory in C (20465)

Assignments for the **2026B** semester at the Open University by Ido Sheffi

## 📂 Project Structure

This repository is structured as a monorepo, containing all assignments for the semester. Each program is completely self-contained with its own source code, Makefile, and test inputs.

## 🛠️ Environment & Compilation

This project is developed locally on macOS but is continuously tested against a strict Linux environment to ensure 100% compatibility with the university's grading scripts.

* **IDE:** CLion
* **Compilers:** GCC / Clang
* **Language Standard:** ANSI C (C89/C90)
* **Required Flags:** `-Wall -ansi -pedantic`
* **CI/CD:** Automated build and test pipeline running on Ubuntu 24.04 via GitHub Actions.
